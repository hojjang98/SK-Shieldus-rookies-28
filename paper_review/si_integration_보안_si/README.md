# 🏗️ SI (보안시스템 구축) — 논문 리뷰

이 폴더는 **보안시스템 통합(Security System Integration, SI)** 분야의 주요 논문을 읽고 정리하기 위한 공간이다.  
단순 탐지나 관제 단계를 넘어, **보안 아키텍처 설계·구현·자동화** 관점에서 보안을 바라보는 것을 목표로 한다.

---

## 📚 논문 목록
- A Zero Trust Architecture for Cloud and IoT Systems (IEEE, 2021)  
  *(실제 링크는 IEEE Xplore에서 확인 가능)*

---

## 🧠 목적
- 보안 SI의 핵심 개념(Zero Trust, Cloud Security, Policy-based Access 등)을 체계적으로 이해한다.  
- 클라우드 및 IoT 환경에서의 **보안 설계 원리와 통합 아키텍처 구조**를 학습한다.  
- SOC 중심의 탐지형 보안에서 **SI 중심의 설계형 보안**으로 사고 전환을 시도한다.

---

📌 이 폴더의 모든 내용은 **보안 아키텍처 설계와 통합 원리의 학습**을 목적으로 하며,  
실제 구현 코드나 배포 스크립트는 포함하지 않는다.
