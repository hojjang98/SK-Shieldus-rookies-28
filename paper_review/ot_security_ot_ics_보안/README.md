# 🏭 OT/ICS Security (산업제어시스템 보안) — 논문 리뷰

이 폴더는 **산업제어시스템(ICS)** 및 **운영기술(OT)** 보안 분야의 주요 논문을 읽고 정리하기 위한 공간이다.
실습보다는 개념 이해와 사고 확장을 목표로 하며,
**사이버-물리 공격(Cyber-Physical Attack)**, **IT-OT 융합 보안**, 그리고 **산업 프로토콜 취약점** 연구 흐름을 중심으로 다룬다.

---

## 📚 논문 목록

- To Kill a Centrifuge: A Technical Analysis of What Stuxnet's Creators Tried to Achieve (Langner, 2013)

---

## 🧠 목적

- IT 보안과 OT 보안의 근본적 차이(CIA vs AIC 우선순위)를 이해하고, 산업 환경 특유의 보안 요구사항을 파악한다.
- SCADA, DCS, PLC 등 산업제어시스템의 구조와 Purdue 모델 기반 네트워크 아키텍처를 분석한다.
- Stuxnet을 통해 사이버-물리 공격의 3단계 모델(IT → ICS → Physical)을 이해하고, 물리적 피해로 이어지는 공격 체인을 학습한다.
- Air Gap 신화의 한계를 인식하고, IEC 62443 / NIST SP 800-82 기반의 심층 방어 사고를 확립한다.

---

> 📌 이 폴더의 모든 내용은 논문을 통한 **산업제어시스템의 보안 원리 및 위협 분석**을 목적으로 하며,
> 특정 벤더(Siemens, Rockwell 등)의 제품 설정이나 상세 프로토콜 구현은 포함하지 않는다.
