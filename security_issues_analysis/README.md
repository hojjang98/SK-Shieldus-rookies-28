# 주간 보안 이슈 분석 📰🔒

SK쉴더스 본사 방문 이후 시작한 보안 시사 학습 프로젝트입니다.
매주 1~3개의 보안 관련 기사를 선정하여 분석하고, 근본 원인과 예방 방법을 고민합니다.

## 📌 프로젝트 목적

- 최신 보안 트렌드 및 이슈 파악
- 보안 사고의 근본 원인 분석 능력 향상
- 실무적인 보안 대응 방안 학습
- 지속적인 보안 감각 유지

## 📝 분석 방식

각 기사는 다음 항목으로 분석합니다:

1. **기사 정보**: 제목, 출처, 날짜, 링크
2. **핵심 요약**: 2-3줄로 정리
3. **사건/이슈 배경**: 무슨 일이 일어났는가?
4. **원인 분석**: 왜 이런 일이 발생했는가?
5. **영향 및 파급효과**: 어떤 영향을 미쳤는가?
6. **예방 및 대응 방안**: 어떻게 막을 수 있었는가?
7. **개인 인사이트**: 배운 점, 느낀 점
